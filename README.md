## Featured Projects

### 🎮 [Behavioral Playtime Analytics: The PLAICraft Project]
* **Core Stack:** `R` (`tidyverse`, `ggplot2`, `broom`)
* **Description:** Analyzed player acquisition, demographic profiles, and user session telemetry logs generated by a custom Minecraft server deployed by a UBC Computer Science research group. Implemented regression modeling to isolate high-density player engagement segments for targeted recruitment.

### 🌎 [NASA EONET API Natural Event Tracker]
* **Core Stack:** `R` (`httr`, `jsonlite`, `tidyverse`, `ggplot2`)
* **Description:** Engineered an automated data pipeline querying NASA’s Earth Observatory Natural Event Tracker (EONET) REST API. Developed flattening algorithms using `tidyverse` to parse complex, multi-dimensional JSON spatial geometry vectors into clean, relational datasets for global natural event mapping.

### 📊 [Web Scraping & HTML Parsing Engine]
* **Core Stack:** `R` (`rvest`, `xml2`, `tidyverse`)
* **Description:** Built a robust web extraction engine designed to crawl e-commerce sandbox environments ethically by verifying `robots.txt` compliance. Parsed and cleaned raw, semi-structured HTML components (item titles, dynamic pricing nodes, star ratings) into analysis-ready tibble dataframes.
